## Краткое решение

Игра строится вокруг старой орбитальной коммуналки, которую игрок расширяет в уютный космический жилой комплекс. Игрок зарабатывает кредиты с аренды и сервисов, покупает модули, повышает комфорт, заселяет необычных жильцов и делает prestige через репутацию станции.

Кодовая база должна быть одна. Desktop и mobile отличаются компоновкой, плотностью интерфейса и размером интерактивных элементов, но используют одинаковые данные, экономику, сохранения и игровой цикл. Техническое решение: React отвечает за UI, PixiJS отвечает только за живую сцену станции.

Визуальный стиль: **Retro Soviet Space Cozy**. Это теплая бытовая sci-fi эстетика с округлыми модулями, эмалевыми панелями, ламповыми индикаторами, табличками и светящимися окнами. Стиль должен оставаться дружелюбным casual, без мрачного реализма и перегруза деталями.

Content/progression roadmaps описывают будущий scope, но не расширяют MVP. Первый билд остается компактным и должен только оставить технические точки расширения для будущих модулей, жильцов, prestige tiers и Pixi hardpoints.

## Главные принципы

- Первый прогресс должен быть виден за 30-60 секунд.
- Игрок всегда должен понимать, что купить дальше.
- Реклама дает ускорение, но не блокирует базовый прогресс.
- Уютно-комедийный тон важнее злой сатиры.
- Визуальная станция должна расти вместе с экономикой.
- Retro Soviet Space Cozy используется как единый визуальный ориентир для UI, станции, промо и ассетов.
- Mobile layout проектируется как основной для коротких сессий.
- Desktop layout показывает больше информации и лучше подходит для сравнения апгрейдов.
